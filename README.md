<h3>Description:</h3>
<body>The purpose of this project is to create an internal network via Virtualbox where traffic from a Windows 11 machine is monitored, and malicious activity is prevented. Through the use of Snort and Splunk, logs will be collected by the IDS and forwarded using a Splunk Forwarder to an external SIEM server for further threat analysis.</body>


<h2>Utilities Used</h2>
<b>
- Splunk</br>
- Snort
</b>
<h2>Environments Used </h2>
<b>
- VM1 Kali Linux</br>
- VM2 Windows 11</br>
- VM3 Ubuntu w/ Snort and Splunk Forwarder</br>
- VM4 Ubuntu Splunk Server</br>
</b>
